AI Data Retrieval and Ranking Query Design

Step 1: Create the Table

CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    score INT,
    passed BOOLEAN,
    class INT
);

STEP 3: SQL Query Tasks

1. Retrieve all columns from the students table.
Ans - SELECT * FROM students;

2. Retrieve only the name and score columns.
Ans - SELECT name,score From students;

3. Retrieve students who scored above 80.
Ans - SELECT * FROM students
      WHERE score > 80;

4. Retrieve students who scored above 80 AND passed.
Ans - SELECT * FROM students
      WHERE score > 80 AND passed = TRUE;

5. Retrieve students who scored above 85 OR belong to class 9.
Ans - SELECT * FROM students
      WHERE score > 85 OR class = 9;

6. Retrieve students who did NOT pass the exam.
Ans - SELECT * FROM students
      WHERE NOT passed = TRUE;

7. Return a list of distinct classes.
Ans - SELECT DISTINCT class FROM students;

8. Retrieve the top 5 highest-scoring students.
Ans - SELECT * FROM students
      ORDER BY score DESC
      LIMIT 5;

9. Retrieve students sorted first by class (ascending) and then by score (descending).
Ans - SELECT * FROM students
      ORDER BY class ASC, score DESC;

10. Use an alias to rename score as final_score.
Ans - SELECT score AS final_score FROM students;

11. Write one combined query that:
    A. Selects name and score
    B. Filters students who scored above 75
    C. Sorts by score in descending order
    D. Returns only the top 3 results
Ans - SELECT name,score FROM students
      WHERE score > 75
      ORDER BY score DESC
      LIMIT 3;
